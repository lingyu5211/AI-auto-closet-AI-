🧩 核心组件

组件 类型 职责
TryOnCanvas 新组件 通用 Canvas 渲染引擎，加载模特底图 + 叠加衣物图层，支持导出
OutfitPreview 新组件 智能推荐卡片中的穿搭预览，内嵌 TryOnCanvas 静态模式
DressingRoom 新组件 虚拟试衣间交互层，衣物选择 + 拖拽调整 + 层级控制
ModelSwitch 新组件 男/女模特切换按钮
OutfitCard 改造 卡片增加 OutfitPreview 预览区域

📊 数据流

智能推荐流程：
wardrobe.clothes → matchStore.generateSmartOutfits()
  → Outfit[] (含 clothes ids)
  → OutfitCard → OutfitPreview → TryOnCanvas
    → 加载模特底图 (根据 user.gender / 手动选择)
    → 按衣物 type 分层渲染到 Canvas
    → 显示穿搭预览图

手动试穿流程：
用户选择模特 → 选择衣物 → DressingRoom
  → TryOnCanvas 实时更新
  → 拖拽调整位置/大小/层级
  → 保存穿搭 → matchStore.addOutfit()
  → 可导出 Canvas 为图片
