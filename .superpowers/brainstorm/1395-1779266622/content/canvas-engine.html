🔧 TryOnCanvas 组件 API 设计

Props 类型 说明
modelSrcstring模特图片路径
clothesClothing[]要渲染的衣物列表
interactiveboolean是否可拖拽交互（预览=false，试衣间=true）
canvasWidthnumber画布宽度
canvasHeightnumber画布高度
Events
@update(clothes)衣物位置/大小变更时触发
@export(dataURL)导出穿搭图片
Methods
addClothing(item)添加一件衣物到画布
removeClothing(id)移除一件衣物
exportImage()导出当前画布为图片
reset()清空所有衣物

📝 关键实现细节

  • 衣物图片缩放适配：渲染时按衣物类型预设比例缩放（如上衣宽度=模特肩宽×0.9）
  • 层级自动管理：根据 clothing.type 自动分配 z-index，内搭→上衣→外套→配饰
  • 静态/交互双模式：preview 模式纯展示，dressing 模式支持触摸拖拽+缩放
  • 导出功能：canvas.toDataURL() 导出完整穿搭图，可保存到相册或分享
  • 模特切换：切换男/女模时保留已选衣物，重新渲染到新模特上
